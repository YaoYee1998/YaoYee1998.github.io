Postman入门教程

Postman是google开发一款Chrome插件，用于发送网页HTTP请求，高阶程度可以运行测试用例。

  • 下载和使用

    下载需用通过翻墙到谷歌的应用拓展里面搜索并且下载，不仅仅可以通过网页插件使用，还可以建立本地的快捷图标。（公司的wifi应该是可以自动翻墙的，不可以翻墙的话应该可以自己下载安装包手动安装）

  • 主要功能

    • 模拟各种HTTP requests
      增删改查：GET、POST、 PUT 、 DELETE
    • Collection 功能（测试集合）
      Collection 是 requests的集合，存储一次测试所用request可以方便多次使用。Collection 可 Import 可 Share 。
    • Response整理
      Postman 可以针对response内容的格式自动美化。 JSON、 XML 或是 HTML 都整理成我们可以阅读的格式。（类似于编辑器，可以选择不同的语言格式）
    • 内置测试脚本语言
      Postman支持编写测试脚本，可以快速的检查request的结果，并返回测试结果

  • 新建首页的接口（右键add request），对应第一个分级界面
